# CLI

The CLI follows the open-code registry model: it fetches component definitions and writes source files into your app.

## init

```bash
npx shadcn-lynx init
```

## add

```bash
npx shadcn-lynx add switch
```

Multiple items can be added in one command.

```bash
npx shadcn-lynx add button input form
```
