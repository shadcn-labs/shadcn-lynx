# Dark Mode

Lynx hosts can provide theme information through global properties. Pair that host state with CSS variables or theme classes in your app root.

```tsx
const appTheme = lynx.__globalProps?.appTheme;

<view className={appTheme === 'dark' ? 'dark' : ''}>
  {children}
</view>
```

Keep the component files theme-token based so they do not need platform-specific branches.
