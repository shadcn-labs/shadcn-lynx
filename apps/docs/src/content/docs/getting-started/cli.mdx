# CLI

The shadcn-lynx CLI helps you add and manage components in your project.

## Commands

### init

Initialize your project for shadcn-lynx:

```bash
npx shadcn-lynx init
```

This creates:
- `components.json` configuration
- `lib/utils.ts` with the `cn` utility
- Base component structure

### add

Add components to your project:

```bash
# Add a single component
npx shadcn-lynx add button

# Add multiple components
npx shadcn-lynx add card input label

# Add all components
npx shadcn-lynx add --all
```

Components are copied to your `components/ui` directory.

### diff

Check for component updates:

```bash
npx shadcn-lynx diff button
```

### update

Update existing components:

```bash
npx shadcn-lynx update button
```

## Configuration

The `components.json` file controls how components are added:

```json
{
  "style": "default",
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/styles/globals.css",
    "baseColor": "neutral",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils"
  }
}
```

### Options

- **style**: The component style variant
- **tailwind.config**: Path to your Tailwind config
- **css**: Path to your global CSS
- **baseColor**: The base color palette
- **cssVariables**: Use CSS variables for theming
- **aliases.components**: Import alias for components
- **aliases.utils**: Import alias for utilities
