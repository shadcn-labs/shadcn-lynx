# Installation

Learn how to set up shadcn-lynx in your Lynx project.

## Prerequisites

Before you begin, ensure you have:

- Node.js 18 or later
- pnpm package manager
- A ReactLynx project with Tailwind CSS configured

## Initialize Your Project

Run the following command in your project root:

```bash
npx shadcn-lynx init
```

This will:

1. Create a `components.json` configuration file
2. Set up the utility files (`cn` function, etc.)
3. Configure the component import paths

## Manual Setup

If you prefer to set up manually, create the following files:

### components.json

```json
{
  "style": "default",
  "tailwind": {
    "config": "tailwind.config.js",
    "css": "src/styles/globals.css",
    "baseColor": "neutral",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils"
  }
}
```

### lib/utils.ts

```typescript
import { type ClassValue, clsx } from 'clsx'
import { twMerge } from 'tailwind-merge'

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```
