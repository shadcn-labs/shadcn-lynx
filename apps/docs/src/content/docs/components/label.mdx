# Label

Renders an accessible label associated with controls.

## Installation

```bash
npx shadcn-lynx add label
```

## Usage

```typescript


export function App() {
  return (
    <view>
      <Label>Email</Label>
    </view>
  )
}
```

import { ComponentPreview } from '@/components/component-preview'
import { LabelDefaultDemo, LabelWithInputDemo } from '@/examples/label-demo'


## Examples

### Default

<ComponentPreview
  component={LabelDefaultDemo}
  code={`<Label>Default Label</Label>`}
/>

### With Input

<ComponentPreview
  component={LabelWithInputDemo}

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `TextProps` | - | Native text props |
