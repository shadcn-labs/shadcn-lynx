# Dialog

A window overlaid on either the primary window or another dialog window.

## Installation

```bash
npx shadcn-lynx add dialog
```

## Usage

```typescript
import {
  DialogRoot,
  DialogTrigger,
  DialogClose,
  DialogBackdrop,
  DialogContent,
  DialogHeader,
  DialogFooter,
  DialogTitle,
  DialogDescription,
} 

export function App() {
  return (
    <DialogRoot>
      <DialogTrigger>Edit Profile</DialogTrigger>
      <DialogBackdrop />
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Edit profile</DialogTitle>
          <DialogDescription>
            Make changes to your profile here.
          </DialogDescription>
        </DialogHeader>
        <text>Profile content</text>
        <DialogFooter>
          <DialogClose>Cancel</DialogClose>
          <DialogClose>Save changes</DialogClose>
        </DialogFooter>
      </DialogContent>
    </DialogRoot>
  )
}
```

## API Reference

### DialogRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| show | `boolean` | - | The controlled open state |
| defaultShow | `boolean` | `false` | The default open state |
| onShowChange | `(open: boolean) => void` | - | Event handler for open changes |
| onOpen | `() => void` | - | Open event handler |
| onClose | `() => void` | - | Close event handler |
| forceMount | `boolean` | `false` | Force mount content |
| ...props | `DialogRootProps` | - | Native dialog props from @lynx-js/lynx-ui |

### DialogTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| disabled | `boolean` | `false` | Whether the trigger is disabled |
| ...props | `DialogTriggerProps` | - | Native trigger props |

### DialogContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `DialogContentProps` | - | Native content props |

### DialogHeader Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|

| ...props | `ViewProps` | - | Native view props |

### DialogFooter Props
