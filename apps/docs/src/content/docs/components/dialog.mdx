# Dialog

A window overlaid on either the primary window or another dialog window.

## Installation

```bash
npx shadcn-lynx add dialog
```

## Usage

```typescript
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogFooter,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from '@/components/ui/dialog'

export function App() {
  return (
    <Dialog>
      <DialogTrigger>
        <Button>Edit Profile</Button>
      </DialogTrigger>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Edit profile</DialogTitle>
          <DialogDescription>
            Make changes to your profile here.
          </DialogDescription>
        </DialogHeader>
        <view>
          <text>Profile content</text>
        </view>
        <DialogFooter>
          <Button>Save changes</Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  )
}
```

## API Reference

### Dialog Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| open | `boolean` | - | The controlled open state |
| defaultOpen | `boolean` | `false` | The default open state |
| onOpenChange | `(open: boolean) => void` | - | Event handler for open changes |
| ...props | `DialogProps` | - | Native dialog props from @lynx-js/lynx-ui |

### DialogTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| asChild | `boolean` | `false` | Render as child component |
| ...props | `ViewProps` | - | Native view props |

### DialogContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |
