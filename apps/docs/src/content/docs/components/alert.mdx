# Alert

Displays a callout for user attention.

## Installation

```bash
npx shadcn-lynx add alert
```

## Usage

```typescript
import { Alert, AlertTitle, AlertDescription } 

export function App() {
  return (
    <Alert>
      <AlertTitle>Heads up!</AlertTitle>
      <AlertDescription>
        You can add components to your project using the CLI.
      </AlertDescription>
    </Alert>
  )
}
```

import { ComponentPreview } from '@/components/component-preview'
import { AlertDefaultDemo, AlertDestructiveDemo } from '@/examples/alert-demo'

## API Reference

### Alert Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `default` \| `destructive` | `default` | The visual variant |
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### AlertTitle Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `TextProps` | - | Native text props |

### AlertDescription Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `TextProps` | - | Native text props |
