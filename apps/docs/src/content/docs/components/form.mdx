# Form

Building forms with built-in validation and error handling.

## Installation

```bash
npx shadcn-lynx add form
```

## Usage

```typescript
import {
  FormRoot,
  FormField,
  FormItem,
  FormLabel,
  FormControl,
  FormMessage,
  FormSubmitButton,
} 


export function App() {
  return (
    <FormRoot
      initialValues={{ email: '' }}
      onSubmit={(values) => console.log(values)}
    >
      <FormField name="email">
        <FormItem>
          <FormLabel>Email</FormLabel>
          <FormControl>
            <Input placeholder="Enter your email" />
          </FormControl>
          <FormMessage />
        </FormItem>
      </FormField>
      <FormSubmitButton>Submit</FormSubmitButton>
    </FormRoot>
  )
}
```

## API Reference

### FormRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| initialValues | `object` | - | Initial form values |
| onSubmit | `(data: any) => void` | - | Submit handler |
| onChanged | `(values: any) => void` | - | Change handler |
| className | `string` | - | Additional CSS classes |
| ...props | `FormRootProps` | - | Native form props from @lynx-js/lynx-ui |

### FormField Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| name | `string` | - | The field name |
| ...props | `FormFieldProps` | - | Native field props |

### FormItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### FormLabel Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `TextProps` | - | Native text props |

### FormControl Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### FormMessage Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `TextProps` | - | Native text props |

### FormSubmitButton Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| onSubmit | `() => void` | - | Submit handler |
| ...props | `FormSubmitButtonProps` | - | Native button props |
