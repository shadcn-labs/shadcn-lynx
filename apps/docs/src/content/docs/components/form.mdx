# Form

Building forms with built-in validation and error handling.

## Installation

```bash
npx shadcn-lynx add form
```

## Usage

```typescript
import {
  Form,
  FormField,
  FormItem,
  FormLabel,
  FormControl,
  FormDescription,
  FormMessage,
} from '@/components/ui/form'
import { Input } from '@/components/ui/input'
import { Button } from '@/components/ui/button'

export function App() {
  const form = useForm({
    defaultValues: {
      email: '',
    },
  })

  function onSubmit(data) {
    console.log(data)
  }

  return (
    <Form form={form} onSubmit={onSubmit}>
      <FormField
        control={form.control}
        name="email"
        render={({ field }) => (
          <FormItem>
            <FormLabel>Email</FormLabel>
            <FormControl>
              <Input placeholder="Enter your email" {...field} />
            </FormControl>
            <FormDescription>
              We'll never share your email.
            </FormDescription>
            <FormMessage />
          </FormItem>
        )}
      />
      <Button type="submit">Submit</Button>
    </Form>
  )
}
```

## API Reference

### Form Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| form | `FormInstance` | - | The form instance |
| onSubmit | `(data: any) => void` | - | Submit handler |
| ...props | `ViewProps` | - | Native view props |

### FormField Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| control | `Control` | - | The form control |
| name | `string` | - | The field name |
| render | `RenderFunction` | - | Render function |

### FormItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### FormLabel Props

| Prop | Type | Default | Description |

### FormControl Props
