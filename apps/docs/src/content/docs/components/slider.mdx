# Slider

An input where the user selects a value from within a given range.

## Installation

```bash
npx shadcn-lynx add slider
```

## Usage

```typescript
import {
  SliderRoot,
  SliderTrack,
  SliderRange,
  SliderThumb,
} 

export function App() {
  return (
    <SliderRoot defaultValue={[50]} max={100} step={1}>
      <SliderTrack>
        <SliderRange />
      </SliderTrack>
      <SliderThumb />
    </SliderRoot>
  )
}
```

import { ComponentPreview } from '@/components/component-preview'

## API Reference

### SliderRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `number[]` | - | The controlled value |
| defaultValue | `number[]` | - | The default value |
| min | `number` | `0` | The minimum value |
| max | `number` | `100` | The maximum value |
| step | `number` | `1` | The step interval |
| disabled | `boolean` | `false` | Whether the slider is disabled |
| ...props | `SliderRootProps` | - | Native slider props from @lynx-js/lynx-ui |

### SliderTrack Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### SliderRange Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |

### SliderThumb Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |
