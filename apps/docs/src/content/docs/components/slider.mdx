# Slider

An input where the user selects a value from within a given range.

## Installation

```bash
npx shadcn-lynx add slider
```

## Usage

```typescript
import { Slider } from '@/components/ui/slider'

export function App() {
  const [value, setValue] = useState([50])
  
  return (
    <view>
      <Slider
        value={value}
        onValueChange={setValue}
        max={100}
        step={1}
      />
    </view>
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `number[]` | - | The controlled value |
| defaultValue | `number[]` | - | The default value |
| min | `number` | `0` | The minimum value |
| max | `number` | `100` | The maximum value |
| step | `number` | `1` | The step interval |
| disabled | `boolean` | `false` | Whether the slider is disabled |
| onValueChange | `(value: number[]) => void` | - | Event handler for value changes |
| className | `string` | - | Additional CSS classes |
| ...props | `SliderProps` | - | Native slider props from @lynx-js/lynx-ui |
