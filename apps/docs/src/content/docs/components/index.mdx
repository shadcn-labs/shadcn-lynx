## Component Pattern

All shadcn-lynx components follow this pattern:

```typescript
import { BaseComponent } from '@lynx-js/lynx-ui'
import { cn } from '@/lib/utils'
import { cva, type VariantProps } from 'class-variance-authority'

const componentVariants = cva('base-classes', {
  variants: {
    variant: { /* ... */ },
    size: { /* ... */ },
  },
  defaultVariants: {
    variant: 'default',
    size: 'default',
  },
})

export function Component({ className, variant, size, ...props }) {
  return (
    <BaseComponent
      className={cn(componentVariants({ variant, size, className }))}
      {...props}
    />
  )
}
```

## Adding Components

Use the CLI to add components:

```bash
npx shadcn-lynx add button
npx shadcn-lynx add card input label
```
