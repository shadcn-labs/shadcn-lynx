# Popover

Displays rich content in a portal, triggered by a button.

## Installation

```bash
npx shadcn-lynx add popover
```

## Usage

```typescript
import {
  PopoverRoot,
  PopoverTrigger,
  PopoverContent,
  PopoverPositioner,
} 

export function App() {
  return (
    <PopoverRoot>
      <PopoverTrigger>Open Popover</PopoverTrigger>
      <PopoverPositioner>
        <PopoverContent>
          <text>Popover content goes here</text>
        </PopoverContent>
      </PopoverPositioner>
    </PopoverRoot>
  )
}
```

## API Reference

### PopoverRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| show | `boolean` | - | The controlled open state |
| defaultShow | `boolean` | `false` | The default open state |
| onVisibleChange | `(visible: boolean) => void` | - | Event handler for open changes |
| onOpen | `() => void` | - | Open event handler |
| onClose | `() => void` | - | Close event handler |
| forceMount | `boolean` | `false` | Force mount content |
| ...props | `PopoverRootProps` | - | Native popover props from @lynx-js/lynx-ui |

### PopoverTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| disabled | `boolean` | `false` | Whether the trigger is disabled |
| ...props | `PopoverTriggerProps` | - | Native trigger props |

### PopoverContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `PopoverContentProps` | - | Native content props |

### PopoverPositioner Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| placement | `string` | - | The placement of the popover |
| ...props | `PopoverPositionerProps` | - | Native positioner props |
