# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.

## Installation

```bash
npx shadcn-lynx add radio-group
```

## Usage

```typescript
import { RadioGroup, RadioGroupItem } from '@/components/ui/radio-group'
import { Label } from '@/components/ui/label'

export function App() {
  const [value, setValue] = useState('option1')
  
  return (
    <RadioGroup value={value} onValueChange={setValue}>
      <view style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
        <RadioGroupItem value="option1" id="option1" />
        <Label htmlFor="option1">Option 1</Label>
      </view>
      <view style={{ flexDirection: 'row', alignItems: 'center', gap: 8 }}>
        <RadioGroupItem value="option2" id="option2" />
        <Label htmlFor="option2">Option 2</Label>
      </view>
    </RadioGroup>
  )
}
```

## API Reference

### RadioGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | - | The controlled value |
| defaultValue | `string` | - | The default value |
| disabled | `boolean` | `false` | Whether the group is disabled |
| onValueChange | `(value: string) => void` | - | Event handler for value changes |
| className | `string` | - | Additional CSS classes |
| ...props | `RadioGroupProps` | - | Native radio group props from @lynx-js/lynx-ui |

### RadioGroupItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | - | The value of the radio button |
| disabled | `boolean` | `false` | Whether the item is disabled |
| className | `string` | - | Additional CSS classes |
| ...props | `RadioGroupItemProps` | - | Native radio item props from @lynx-js/lynx-ui |
