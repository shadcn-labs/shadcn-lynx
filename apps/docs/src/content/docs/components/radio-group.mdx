# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time.

## Installation

```bash
npx shadcn-lynx add radio-group
```

## Usage

```typescript
import { RadioGroupRoot, RadioItem } 

export function App() {
  return (
    <RadioGroupRoot defaultValue="option1">
      <RadioItem value="option1" label="Option 1" />
      <RadioItem value="option2" label="Option 2" />
      <RadioItem value="option3" label="Option 3" />
    </RadioGroupRoot>
  )
}
```

## API Reference

### RadioGroupRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | - | The controlled value |
| defaultValue | `string` | - | The default value |
| disabled | `boolean` | `false` | Whether the group is disabled |
| onValueChange | `(value: string) => void` | - | Event handler for value changes |
| className | `string` | - | Additional CSS classes |
| ...props | `RadioGroupRootProps` | - | Native radio group props from @lynx-js/lynx-ui |

### RadioItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | - | The value of the radio button |
| disabled | `boolean` | `false` | Whether the item is disabled |
| label | `string` | - | Label text |
| className | `string` | - | Additional CSS classes |
| ...props | `RadioProps` | - | Native radio item props from @lynx-js/lynx-ui |
