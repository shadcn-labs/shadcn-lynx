# Button

Displays a button or a component that looks like a button.

## Installation

```bash
npx shadcn-lynx add button
```

## Usage

```typescript
import { Button } from '@/components/ui/button'

export function App() {
  return (
    <view>
      <Button>Click me</Button>
    </view>
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `default` \| `destructive` \| `outline` \| `secondary` \| `ghost` \| `link` | `default` | The visual variant |
| size | `default` \| `sm` \| `lg` \| `icon` | `default` | The size variant |
| disabled | `boolean` | `false` | Whether the button is disabled |
| className | `string` | - | Additional CSS classes |
| onClick | `() => void` | - | Click handler |
| ...props | `ButtonProps` | - | Native button props from @lynx-js/lynx-ui |
