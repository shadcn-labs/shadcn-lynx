# Sheet

Extends the Dialog component to display content that complements the main content of the view.

## Installation

```bash
npx shadcn-lynx add sheet
```

## Usage

```typescript
import {
  SheetRoot,
  SheetTrigger,
  SheetClose,
  SheetContent,
  SheetHeader,
  SheetFooter,
  SheetTitle,
  SheetDescription,
  SheetBackdrop,
} 

export function App() {
  return (
    <SheetRoot>
      <SheetTrigger>Open Sheet</SheetTrigger>
      <SheetBackdrop />
      <SheetContent side="right">
        <SheetHeader>
          <SheetTitle>Edit profile</SheetTitle>
          <SheetDescription>
            Make changes to your profile here.
          </SheetDescription>
        </SheetHeader>
        <text>Sheet content</text>
        <SheetFooter>
          <SheetClose>Close</SheetClose>
        </SheetFooter>
      </SheetContent>
    </SheetRoot>
  )
}
```

## API Reference

### SheetRoot Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| show | `boolean` | - | The controlled open state |
| defaultShow | `boolean` | `false` | The default open state |
| onShowChange | `(open: boolean) => void` | - | Event handler for open changes |
| side | `top` \| `right` \| `bottom` \| `left` | `right` | The side to display from |
| ...props | `SheetRootProps` | - | Native sheet props from @lynx-js/lynx-ui |

### SheetTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |
