# Badge

Displays a badge or a component that looks like a badge.

## Installation

```bash
npx shadcn-lynx add badge
```

## Usage

```typescript
import { Badge } from '@/components/ui/badge'

export function App() {
  return (
    <view>
      <Badge>Badge</Badge>
    </view>
  )
}
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `default` \| `secondary` \| `outline` \| `destructive` | `default` | The visual variant |
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |
