# Badge

Displays a badge or a component that looks like a badge.

## Installation

```bash
npx shadcn-lynx add badge
```

## Usage

```typescript
import { Badge } 

export function App() {
  return (
    <view>
      <Badge>Badge</Badge>
    </view>
  )
}
```

import { ComponentPreview } from '@/components/component-preview'
import { BadgeDefaultDemo, BadgeVariantsDemo } from '@/examples/badge-demo'

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `default` \| `secondary` \| `outline` \| `destructive` | `default` | The visual variant |
| className | `string` | - | Additional CSS classes |
| ...props | `ViewProps` | - | Native view props |
